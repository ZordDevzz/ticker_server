跑馬燈進階系統 (Ticker Pro) 使用手冊

本系統是專為數位看板 (Digital Signage)、直播疊加 (OBS/Streaming) 及會議室通知設計的專業級跑馬燈解決方案。支援跨設備同步、離線持久化顯示及全自動連接恢復。

1. 核心功能亮點

📡 跨設備遠端控制

透過任何手機或電腦的控制面板，即時更新所有連網螢幕的內容。

💾 離線持久化 (Standalone)

顯示端會自動記憶最後接收的內容。即使斷網或伺服器重啟，螢幕仍會持續播放，不會留白。

🎥 直播透明背景

一鍵切換透明背景，可直接作為 OBS 瀏覽器來源 (Browser Source)，完美疊加在影片上。

🎯 精準設備管理

自動識別不同螢幕的解析度與設備類型，支援個別發送訊息或全體廣播。

2. 操作指南

A. 伺服器啟動 (Server)

  • 確保已安裝 Node.js。
  • 在專案資料夾執行：node server.js
  • 預設埠位為 3000。若要修改請編輯 server.js 中的 PORT 常數。

B. 控制面板 (Control Panel)

  • 位址： http://[伺服器IP]:3000
  • 設定記憶： 面板會自動保存您的配色、語系與偏好，下次開啟時無需重新設定。
  • 語系切換： 右上角可切換 English 或 繁體中文，面板所有欄位將同步翻譯。

C. 顯示端 (Receiver / Signage)

  • 位址： http://[伺服器IP]:3000/display
  • 狀態燈： 右下角的小圓點代表連線狀態：
    ● 綠色： 線上同步中。
    ● 紅色： 離線模式（正顯示最後一次存儲的內容，並自動嘗試重連）。

3. 進階參數說明

  • 捲動速度： 範圍 2-60 秒。數值越小，文字捲動越快。建議長文本設為 20-30 秒。
  • 垂直位置：
    • Top: 位於畫面頂端 10% 處。
    • Middle: 畫面中心 (50%)。
    • Bottom: 畫面底端 90% 處。
  • 透明背景： 勾選後，背景色會完全消失，僅保留文字及陰影，適合疊加層使用。
💡 專業建議： 對於 24/7 運行的展示牆，建議將瀏覽器設為「全螢幕模式」(按下 F11)，並關閉作業系統的自動休眠功能。

4. 故障排除 (Troubleshooting)

  • 連不上伺服器？ 請確認控制設備與顯示設備皆在同一個區域網路 (LAN/Wi-Fi)，並檢查伺服器主機的防火牆是否已開啟 3000 埠。
  • 文字重疊或動畫異常？ 本系統採用現代 CSS 動畫，請確保使用最新版的 Chrome、Edge 或 Safari 瀏覽器。
